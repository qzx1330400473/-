## 如何访问

* 下载
    ```
       git clone https://github.com/qzx1330400473/future-home.git
    ```
   
* 进入目录
    ```
        cd yuetao
    ```

* 安装依赖
    ```
        npm install
    ```
    
* 启动项目
    ```
        node server.js
    ```
    
* 访问
    ```
        http://127.0.0.1:16931/文件夹/文件夹名
    ```
